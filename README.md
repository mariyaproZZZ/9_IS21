Описание
Проект состоит из трёх модулей:
- Модуль A: генератор случайных паролей
- Модуль B: оценка сложности паролей
- Модуль C: шифрование и сохранение паролей

Инструкция по запуску
1. Клонировать репозиторий
2. Запустить нужный модуль: `python 9_modul_*.py`
